# calculator
A small little "hello world" built with next.js and react

## To run this app do the following steps:

Clone the repo:  
`git clone git@github.com:Ojvind/calculator.git`

Install all the dependencies:  
`npm install`

...or if you (like me) prefer yarn:  
`yarn install`

Start the app:  
`yarn dev`  
or `npm run dev`

Browse to:  
`http://localhost:3000`
